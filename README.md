# Flipkart Customer Support Project

## 📌 Overview
This project analyzes Flipkart customer support data and builds a machine learning model to predict customer satisfaction (CSAT) based on customer interactions, queries, and feedback.

---

## 🎯 Objective
- Analyze customer support chat/data
- Predict Customer Satisfaction Score (CSAT)
- Improve customer service quality using ML insights

---

## 🧠 Problem Statement
Customer support teams receive large volumes of queries. This project helps in understanding and predicting customer satisfaction to improve response quality and efficiency.

---

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Jupyter Notebook

---

## 📊 Workflow
1. Data Collection  
2. Data Cleaning  
3. Exploratory Data Analysis (EDA)  
4. Feature Engineering  
5. Model Building  
6. Model Evaluation  
7. Prediction
