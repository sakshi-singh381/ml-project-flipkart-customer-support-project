# 📊 Flipkart Customer Support Project (CSAT Prediction)

## 📌 Overview
This project focuses on analyzing Flipkart customer support data and building a Machine Learning model to predict **Customer Satisfaction (CSAT)** based on customer interactions, feedback, and issue details.

---

## 🎯 Problem Statement
E-commerce platforms like Flipkart handle thousands of customer queries daily.  
The challenge is to identify factors affecting customer satisfaction and predict CSAT scores to improve customer experience.

---

## 🧠 Objective
- Analyze customer support data
- Identify patterns affecting satisfaction
- Build ML model to predict CSAT
- Improve customer service insights

---

## 📂 Dataset Information
- Customer support interaction data
- Includes:
  - Issue type
  - Response time
  - Customer feedback
  - CSAT score
- Source: Internal/Kaggle dataset (if used, mention exact link)
